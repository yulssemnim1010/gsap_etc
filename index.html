Vibe Making Process

I start with a visual mood, organize the experience into clear sections, and add motion where it makes the interface feel more intentional.

01

Find the Mood

I collect references, colors, textures, and keywords to define the emotional direction before designing the first screen.

02

Build the Flow

I structure pages with clear sections, readable hierarchy, and responsive layouts so the content works across different screen sizes.

03

Add Motion Detail

I use scroll, hover, and transition details to guide attention without making the interface feel noisy.
